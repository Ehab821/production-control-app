Select Project Number and Pod Type. The Work Order No. is loaded from imported Oracle slab casting work orders. If one open WO exists, it is auto-selected; if multiple open WOs exist, the user must select the correct one; if no open WO exists, casting is blocked. Slab Release & Delivery

Workflow: Daily Demand → Casting Team confirms shipment → Production Supervisor confirms delivery → Work orders are generated for delivered qty only.

Pending Shipment - Casting Team

Requested Qty is read-only. Casting team updates Ship Qty Now, selects lines, then confirms shipment once for selected lines. Stock reduces at shipment.

Pending Delivery - Production Supervisor

Supervisor updates Deliver Qty Now, selects lines, then confirms delivery. Work orders are generated only for delivered qty. Any delivery difference creates a review issue.
